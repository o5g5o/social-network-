# social-network

## dependencies:
- npm install 

- npm install tailwindcss @tailwindcss/vite 

- npm install axios (we are going to use this instead of the normal fetch so learn it XD )

- npm install @tanstack/react-query


## run:
1. Open two terminals

2. First Terminal:
- for frontend 
    - cd frontend
    - npm run dev

3. Second Terminal:
- for backend
    - cd backend
    - go run .
